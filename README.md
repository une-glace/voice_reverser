# 倒放语音助手 (Voice Reverser)

这是一个简单有趣的网页应用，可以录制你的声音并立即倒放出来。完全在浏览器本地运行，无需后台服务器，安全、快速且免费。

## 功能特点

*   **🎤 即时录音**：一键点击开始/停止录音。
*   **◀️ 倒放音频**：录制完成后，立即生成倒放版本，体验奇妙的“反向说话”效果。
*   **▶️ 原声对比**：随时播放原始录音，方便与倒放效果进行对比。
*   **🎧 设备选择**：支持手动选择麦克风设备（如蓝牙耳机、USB麦克风等），解决系统默认设备不准确的问题。
*   **🔒 隐私安全**：纯前端实现，所有音频数据仅在本地浏览器处理，不会上传到任何服务器。

### 本地运行

1.  下载本项目代码。
2.  由于浏览器的安全策略（麦克风权限），直接双击打开 `index.html` 可能无法正常录音。建议使用本地服务器运行。
3.  如果你安装了 Python，可以在项目目录下运行：
    ```bash
    python -m http.server 8000
    ```
4.  在浏览器打开 `http://localhost:8000` 即可使用。

## 技术栈

*   HTML5 / CSS3
*   JavaScript (ES6+)
*   Web Audio API (AudioContext)
*   MediaStream Recording API
